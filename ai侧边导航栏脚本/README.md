# AI 对话侧边导航栏（GPT / Gemini）

一个面向 **ChatGPT** 与 **Gemini** 的油猴脚本。  
提供轻量、稳定、可拖拽缩放的对话目录侧边栏，帮助你在长对话中快速定位与管理内容。

---

## 特性

- **双站点通用**：`chatgpt.com` + `gemini.google.com`
- **目录导航**：自动提取用户提问，点击即滚动定位
- **搜索过滤**：实时检索目录项
- **书签标记**：星标重点问题
- **一键导出**
  - 左键：复制当前目录
  - `Shift + 左键`：复制完整对话
- **高性能设计**
  - 增量渲染
  - 弱引用节点
  - 列表渲染限量
- **交互体验**
  - 可拖动位置
  - 右下角拖动缩放
  - 明暗主题切换
  - Google 简约风视觉

## 部署到油猴（Tampermonkey）

### 1) 安装 Tampermonkey
- Chrome / Edge：在扩展商店搜索 **Tampermonkey** 并安装

### 2) 新建脚本
1. 点击浏览器扩展中的 Tampermonkey 图标  
2. 选择 **创建新脚本**  
3. 删除编辑器内默认模板代码

### 3) 粘贴脚本代码
- 打开项目中的 `侧边栏.js`
- 全选并复制内容
- 粘贴到 Tampermonkey 编辑器
- 点击 **文件 -> 保存**（或 `Ctrl + S`）

### 4) 验证生效
打开以下任一页面并刷新：
- `https://chatgpt.com/`
- `https://gemini.google.com/`

页面右侧会出现侧边导航栏，表示部署成功。

## 使用说明（简版）

- 拖动头部/底部：移动侧边栏
- 拖动右下角：调整宽高
- 点击目录项：滚动到对应消息
- 点击星标：收藏/取消收藏
- 搜索框：过滤目录项
- 底部复制按钮：
  - 左键复制目录
  - `Shift + 左键` 复制完整对话
